A selection of my work on [GitHub](https://github.com/olayasturias), grouped by theme.

## Visual SLAM & Visual Odometry

- [monocular_visual_slam_survey](https://github.com/olayasturias/monocular_visual_slam_survey) — evaluation scripts and data loaders for SOTA SLAM algorithms (companion to my IEEE TAI survey).
- [robot_visual_localization](https://github.com/olayasturias/robot_visual_localization) — ROS packages for robot camera-based localization.
- [gtsam-python-tutorials](https://github.com/olayasturias/gtsam-python-tutorials) — Python tutorials for my talk _"Introduction to GTSAM: Factor Graph Optimization for (Visual) SLAM"_.
- [SLAM-course](https://github.com/olayasturias/SLAM-course) — code examples based on Cyrill Stachniss' SLAM course.
- [awesome_slam_dataset_downloaders](https://github.com/olayasturias/awesome_slam_dataset_downloaders) — list of SLAM datasets plus quick download scripts.
- [dv-ros](https://github.com/olayasturias/dv-ros) — event-camera (DV) ROS utilities.
- [icra2026](https://github.com/olayasturias/icra2026) — code for an ICRA 2026 submission.
- [ORB_SLAM3](https://github.com/olayasturias/ORB_SLAM3) — accurate open-source library for visual, visual-inertial and multi-map SLAM. _(fork)_
- [DROID-SLAM](https://github.com/olayasturias/DROID-SLAM) — deep visual SLAM. _(fork)_
- [dso](https://github.com/olayasturias/dso) — Direct Sparse Odometry. _(fork)_
- [DEVO](https://github.com/olayasturias/DEVO) — [3DV '24] Deep Event Visual Odometry. _(fork)_
- [MAC-VO](https://github.com/olayasturias/MAC-VO) — [ICRA 2025 Best Paper] metrics-aware covariance for learning-based stereo VO. _(fork)_
- [TrianFlow](https://github.com/olayasturias/TrianFlow) — joint depth-pose learning without PoseNet. _(fork)_
- [DF-VO](https://github.com/olayasturias/DF-VO) — Depth and Flow for Visual Odometry. _(fork)_
- [tartanvo](https://github.com/olayasturias/tartanvo) — a generalizable learning-based VO. _(fork)_
- [relpose-gnn](https://github.com/olayasturias/relpose-gnn) — [3DV21] visual camera re-localization using graph neural networks. _(fork)_
- [ESVO_extension](https://github.com/olayasturias/ESVO_extension) — event-based stereo visual odometry extension. _(fork)_
- [orb_slam3_ros_wrapper](https://github.com/olayasturias/orb_slam3_ros_wrapper) — a ROS wrapper for ORB-SLAM3. _(fork)_
- [dso_ros](https://github.com/olayasturias/dso_ros) — ROS wrapper for DSO. _(fork)_
- [rpg_dvs_ros](https://github.com/olayasturias/rpg_dvs_ros) — ROS packages for DVS event cameras. _(fork)_
